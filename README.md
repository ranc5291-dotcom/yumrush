# 🍔 YumRush – AI-Powered Food Delivery System

YumRush is an advanced AI-powered food delivery web application designed to go beyond traditional ordering platforms by integrating intelligent recommendation systems, health-aware filtering, dynamic pricing, and conversational AI.

This project combines full-stack development with applied artificial intelligence to deliver a smarter, more personalized, and engaging food ordering experience.

---

## 🧠 Core Innovation

Unlike traditional platforms, YumRush integrates AI across the entire user journey:

* Personalized food recommendations based on mood and preferences
* Health-aware meal suggestions based on calorie goals and diet types
* Voice-based ordering using speech recognition
* Dynamic pricing based on demand and time
* AI-powered chatbot for real-time support

---

## 🚀 Key Features

### 🤖 AI Food Recommendation System

* Suggests meals based on:

  * Mood (Hungry, Snacky, Sweet craving, etc.)
  * Cuisine preferences
  * Past order history
* Uses scoring-based ranking logic

---

### 🥗 Health-Aware Ordering System

* User defines:

  * Calorie goals
  * Diet type (Keto, Vegan, High Protein, etc.)
  * Fitness goals (Weight loss, Muscle gain)
* System filters and ranks food accordingly

---

### 🎙️ Voice-Based Ordering

* Uses Web Speech API
* Converts voice input → order actions
* Improves accessibility and usability

---

### 🎡 Spin-the-Wheel (Gamification)

* Random food selection system
* Filters:

  * Veg / Non-veg
  * Cheap / Premium
  * Healthy / Spicy / Sweet
* Enhances user engagement

---

### 📈 Dynamic Pricing System

* Prices adjust based on:

  * Peak hours
  * Order demand
  * Time of day
  * Weather conditions
* Implements demand-supply optimization

---

### 💬 AI Chatbot Support

* LLM-based conversational assistant
* Handles:

  * Orders
  * Delivery queries
  * Coupons
  * Platform usage

---

### 📊 Admin Dashboard

* Analytics and insights:

  * Revenue trends
  * Order statistics
  * Peak hours
  * Category-wise sales
* Menu management system
* Review sentiment analysis (Positive / Negative / Neutral)

---

## 🏗️ System Architecture

* Frontend: HTML, CSS, JavaScript
* Backend: FastAPI (REST APIs)
* AI Logic: Rule-based + scoring algorithms
* Deployment: Render (Cloud Platform)
* Chatbot: LLM-based conversational AI
